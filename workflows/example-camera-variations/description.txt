PF-CREATE_EXTRA_CAMERAS workflow. Takes an input image or architectural render and generates 9 cinematic camera and lighting variations of the EXACT same scene in a 3x3 grid. Uses two prompts: a user prompt (JSON with variation parameters) and a system prompt (Camera Variations Generator V3). The LLM output then feeds into two image generation models: Gemini 3 (Nano Banana Pro) and Seedream v4.5 Edit, to compare results from both models. The workflow has 3 sections: INPUT IMAGE, PROMPTING (user prompt + system prompt + Prompt Concatenator), and GENERATION (Nano Banana Pro + Seedream).
